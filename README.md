PYTHON QUIZ APP:
A Simple command-line based quiz Application built using python.
This app asks multiple-choice questions and also calculate the
Final score based on user answers.

FEATURES:
-> Multiple-choice questions(MCQS)
User input system
score calculation
simple and easy to use

TECH STACK:
Python
